## Создание файла `.gitignore`

Файл **.gitignore** указывает Git, какие файлы или директории следует игнорировать.

### Пример содержимого .gitignore:

```gitignore
# Игнорировать директорию node_modules
node_modules/

# Игнорировать временные файлы
*.log
*.tmp

# Игнорировать файлы IDE
.idea/
*.code-workspace

# Игнорировать системные файлы macOS
.DS_Store

# Игнорировать окружение Python
__pycache__/
*.py[cod]

# Игнорировать сборочные артефакты
dist/
build/
*.out
```
### Как использовать этот пример?

1. Создайте файл .gitignore в корневой папке вашего проекта.
2. Вставьте туда содержимое из примера.
3. Git автоматически будет игнорировать указанные файлы и папки.
